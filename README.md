# Copy Monkey Application

## Create .env file
Create a .env file to store your database credentials

```
DB_HOST = localhost
DB_NAME = cm
DB_USER = root
DB_PASS = 
```

## Installation
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
